
### 2️⃣ GitHub Token yarating
1. https://github.com/settings/tokens ga kiring
2. "Generate new token" → "Generate new token (classic)"
3. **Note**: `Bukhari Academy`
4. **Expiration**: `No expiration` (hech qachon tugamaydi)
5. **Scopes**: Faqat `gist` ni belgilang ✅
6. "Generate token" tugmasini bosing
7. Token'ni ko'chiring (faqat 1 marta ko'rinadi!)
   ```
   ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```

### 3️⃣ Saytda sozlang
1. Admin sifatida kiring
2. Sidebar → "Boshqalar" → "Doimiy saqlash"
3. **Gist ID** va **GitHub Token** ni kiriting
4. "Saqlash" tugmasini bosing
5. TAYYOR! ✅
