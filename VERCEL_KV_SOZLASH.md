## ❓ AGAR MUAMMO BO'LSA:

### Vercel KV ko'rinmasa:
1. Vercel Dashboard → Storage
2. Databases ro'yxatida `bukhari-academy-db` borligini tekshiring

### Environment variables yo'q bo'lsa:
1. Vercel Dashboard → bukhari-academy → Settings → Environment Variables
2. Quyidagilar borligini tekshiring:
   - `KV_REST_API_URL`
   - `KV_REST_API_TOKEN`
   - `KV_REST_API_READ_ONLY_TOKEN`
